# ESP32-ArtnetSPI
![Image Description](https://github.com/richard6534/ESP32-ArtnetSPI/blob/main/Image/IMG_0637.JPG)

這款 Artnet ESP32 LED 控制器是一款高效能的開源解決方案，可透過乙太網路或 Wi-Fi 控制可尋址 LED。此專案可讓您使用 Artnet 協議，透過配備 W5500 乙太網路模組或 ESP32-ETH01 的 ESP32 微控制器來控制多個 LED 燈條。程式碼包含一個簡單的 Web 伺服器，用於配置和控制 LED 燈條。為了獲得最佳效能，建議使用 W5500，因為 ETH01 有效能問題。
